# Resnext-DenseNet-Image-Captioning

Code for Image Captioning Model Testing with DenseNet and Resnext CNN Architectures

File Description:

  - The files DenseNetFeatures Notebook and ResNextFearures are used to extract features from the Flickr8K datatset and save them in a pickle file.
  
  - The TextProcessing Notebook is used to preprocess the Flickr8K text files and convert them into a descriptions.txt file.
  
  - ImCapModel Notebook is used to define the Model and Create Train and Test Splits (Just fit the model with the splits)
  
  - Evaluation Notebook is for testing the BLEU Scores of the Model
